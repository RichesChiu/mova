# mova-page

基于 Vite + React + TypeScript + Tailwind CSS 的 Mova 自托管媒体服务器展示页。

## 页面内容

- 主页：展示 Mova 的媒体库、元数据补全、Web 浏览播放能力
- 文档：收拢 API、前端、后端和 crates 文档入口
- 使用方式：展示 Docker Compose 部署和首次媒体库初始化流程

页面内容参考 `/Users/riches/Desktop/mova/README.zh-CN.md`。

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
