## Description

MOVA is a native macOS client for self-hosted media services.

Connect to a MOVA server that you operate or are authorized to use, then manage and watch your media in a focused desktop app. MOVA provides no public streaming service and includes no movies or shows. Your selected server manages your media, accounts, and playback history.

Key features:

• Add and switch between multiple MOVA servers
• Browse libraries, recently added titles, and continue watching
• Search movies, series, and locally available episodes
• Explore seasons, episodes, cast, and media resource details
• Play video natively with selectable audio tracks and subtitles
• Keep per-account playback progress in sync
• Receive library scan updates and server notifications
• Let administrators create users and libraries and monitor scans
• Choose light, dark, or system appearance
• Use the app in Simplified Chinese or English

Requirements:

• macOS 14 or later
• An accessible and compatible MOVA server
• A valid HTTPS configuration is recommended for public servers

Privacy first:

MOVA contains no advertising, cross-app tracking, or third-party analytics. Server settings stay on your Mac, sign-in tokens are stored in the macOS Keychain, and requests go directly to the MOVA server you select.

Learn about MOVA and self-hosting at https://mova.hk/
